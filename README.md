# TicTacToe

Implementation of tic-tac-toe (to revisit using magic square) board game.
  with minimax
  with minimax and alpha-beta pruning
  

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tic_tac_toe'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tic_tac_toe

## Usage

run

```ruby
bin/tic_tac_toe.rb
```
